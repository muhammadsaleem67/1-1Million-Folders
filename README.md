## 📖 About

This repository contains multiple compressed archives with pre-generated folders in different quantities.

The goal of this project is to demonstrate large-scale folder generation for testing, benchmarking, file system experiments, automation practice, and educational purposes.

## 📥 How to Use

1. Download the desired archive.
2. Extract the `.rar` file using WinRAR or 7-Zip.
3. Use the folders for your testing or educational purposes.

## ⚠️ Note

Creating or extracting a very large number of folders (especially **111,110** or **1,000,000**) may take time depending on your computer's performance and available storage.
